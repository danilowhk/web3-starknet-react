## Local Development

- Clone repo \
  `git clone https://github.com/dhruvkelawala/web3-starknet-react`

- Install top-level dependencies \
  `yarn`

- Install sub-dependencies \
  `yarn bootstrap`

- Build and watch for changes \
  `yarn start`
